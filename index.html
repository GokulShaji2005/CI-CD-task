The Poneglyphs of Deployment

The Workflow File

Hidden in .github/workflows/deploy.yml — the ancient text that sets your code free every time you push to main.

.github/workflows/deploy.yml 
name: Deploy to GitHub Pages

# Triggers when you push to main — like setting sail
on:
  push:
    branches:
      - main

permissions:
  contents: read
  pages:    write
  id-token: write

jobs:
  deploy:
    runs-on: ubuntu-latest
    environment:
      name: github-pages
      url:  ${{ steps.deployment.outputs.page_url }}

    steps:
      # ⚓ Zoro finds the code (eventually)
      - name: Checkout code
        uses: actions/checkout@v4

      # 🗺️ Nami sets the course
      - name: Setup Pages
        uses: actions/configure-pages@v4

      # 🎒 Pack the cargo
      - name: Upload artifact
        uses: actions/upload-pages-artifact@v3
        with:
          path: '.'

      # 🚀 Luffy stretches to the finish line
      - name: Deploy to GitHub Pages
        id:   deployment
        uses: actions/deploy-pages@v4
